# Helios - Jarvis AI Voice Chat

A minimal MVP voice chat frontend for Jarvis AI.

## Features

- Voice recognition using Web Speech API
- Text-to-speech responses
- Minimal dark UI
- Real-time conversation display

## Usage

1. Open `index.html` in a modern browser (Chrome/Edge recommended)
2. Click "Start" to begin recording
3. Speak your message
4. Jarvis responds with both text and voice

## Setup

No installation required. Just open `index.html` in a browser.

**Note:** Requires HTTPS (or localhost) for speech recognition to work.
